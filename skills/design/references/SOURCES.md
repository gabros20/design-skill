# Sources and attribution

Purpose: Trace a technique, value, or rule in this pack back to the skill or named author it came
from, with its license — the provenance index for the design pack.

Read when:
- You need to attribute a method, credit an author, or check the license before reusing material.
- A reviewer asks where a specific value, algorithm, or rule originated.

Skip when:
- You are doing design work; this file carries no design guidance, only provenance.

Inputs:
- A technique, value, or claim whose origin you want to confirm.

Produces:
- The originating skill or author and the applicable license.

The pack was composed by mining our own `webdesign` skill plus the external design skills below,
keeping only what `design` owns and rewriting it to the authoring standard. Values and algorithms are
re-expressed, not copied; only sources with a permissive license contribute any verbatim prose, and
only within that license's terms.
